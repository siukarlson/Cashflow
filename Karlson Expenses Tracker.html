Exchange rates (units per £1 GBP)
These rates are used when converting foreign currency assets to GBP. For example if £1 = HKD 9.85, enter 9.85 for HKD. These are saved and used across the app. "Fetch live rates" pulls current cross-rates via the TradingView (RapidAPI) key below — set that first, then come back up here.
Share price updates
The "Update share prices" button and ticker search use TradingView (RapidAPI). Keys are stored locally in your browser only — note that since this app has no backend, keys are visible in the page source, so avoid using a key on a plan you'd mind someone else exhausting.
Subscribe at rapidapi.com/tradingview-data1/api/tradingview-data1 — unofficial reseller, check its quota/pricing before relying on it. Symbol format: EXCHANGE:TICKER (e.g. NASDAQ:AAPL, HKEX:9988, LSE:VOD, ASX:BHP, NZX:AIA) — same as tradingview.com.
